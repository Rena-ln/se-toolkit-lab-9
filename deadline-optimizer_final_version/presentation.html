📅 Assignment Deadline Optimizer

AI-powered personal deadline tracker

Name: Irina Kostina
Email: i.kostina@innopolis.university
Group: CSE-04

🎯 Context

  • End User: University students managing 4–5 courses per semester, each with multiple assignments and exams
  • Problem: Students face overlapping deadlines every week, don't know where to start, panic, and submit late or produce low-quality work
  • Product Idea: AI-powered personal deadline tracker that analyzes your workload, prioritizes tasks by urgency and complexity, and tells you exactly what to work on and when

🔧 Implementation

Tech Stack:

FastAPI PostgreSQL React TypeScript MCP Docker Compose

Version 1: Core prioritization engine — algorithm scores tasks by urgency + weight + status, returns sorted list with AI recommendations

Version 2: Full management UI — create/delete assignments & deadlines, dashboard with charts, conflict detection, MCP tools for AI agent

TA Feedback Addressed: Added management panel for CRUD operations, improved conflict warnings visibility, added productivity statistics
